# AZURA V9 — Image-first reader patch

Qilingan ishlar:
- PDF.js local/CDN loader o‘chirildi.
- Platforma WebP/JPG image-first rejimga o‘tkazildi.
- `js/azura-v9-image-reader.js` qo‘shildi: image lazy/async, fetchpriority, PDF control hide, reader preload.
- `vendor/pdfjs` olib tashlandi.
- Admin/reader matnlari WebP/JPG yo‘nalishiga moslandi.
- Manhwa, manga, novel va komiks platformasi sifatida metadata yangilandi.

Eslatma:
- PDF viewer endi ataylab o‘chirilgan. Boblarni WebP/JPG/PNG rasm sifatida yuklash tavsiya qilinadi.
- Novel matnli readerga ta’sir qilinmadi.
